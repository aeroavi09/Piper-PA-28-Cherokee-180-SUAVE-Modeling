# Piper-PA-28-Cherokee-180-SUAVE-Modeling

The objective was to accurately model and predict high fidelity flight characteristics of a General Aviation aircraft with the SUAVE network.

Utilized class based engineering and OpenVSP for validation.
Implemented a constant-speed propeller model
Implemented a Internal_Combustion_Propeller network with a 180 HP Lycoming O-360

Measurements derived from Owner's Manual of Piper PA-28-180E, FAA Regulations
